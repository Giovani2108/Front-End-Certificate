04 - Grid Template Areas

grid-template-areas permite crear diseños nombrando zonas del grid.
Puedes repetir nombres para expandir un área (como "head head head") o crear varias secciones distintas.
También puedes definir el tamaño de columnas y filas por separado con grid-template-columns y grid-template-rows, sin usar áreas.
