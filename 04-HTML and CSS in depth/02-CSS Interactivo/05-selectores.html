05 - Selectores y Especificidad en CSS

Tipos de Selectores

Los selectores indican a qué elementos HTML se aplican las reglas CSS. Aquí se muestran los más comunes:

Selector Ejemplo Descripción
Universal * Selecciona todos los elementos HTML de la página.
Tipo p Selecciona todas las etiquetas <p> (o cualquier otra etiqueta específica).
Clase .mi-clase Selecciona todos los elementos que tienen la clase "mi-clase". Se usa con punto (.).
ID #mi-id Selecciona un único elemento que tiene el ID "mi-id". Se usa con el símbolo #.
Atributo input[type="text"] Selecciona solo los elementos <input> que tienen el atributo type="text". Por ejemplo, puedes usarlo para aplicar estilos distintos a los campos de texto en un formulario.
Pseudoclase a:hover Selecciona los enlaces (<a>) cuando el cursor del mouse pasa por encima de ellos. Es útil para cambiar el color o subrayado al pasar el mouse (efecto "hover").
Pseudoelemento p::first-line Selecciona solo la primera línea del texto dentro de un párrafo. Por ejemplo, puedes hacer que la primera línea se vea en negrita o de otro color.
Combinado div p.mi-clase Selecciona los elementos <p> que tienen la clase "mi-clase" solo si están dentro de un <div>. Es un selector más específico que combina varios niveles: etiqueta, jerarquía y clase.

Jerarquía de Especificidad

La especificidad es la forma que tiene CSS de decidir qué reglas aplicar cuando hay conflicto. Esta es la jerarquía:

  1. Estilos en línea: style="..."
  2. IDs: #elemento
  3. Clases, atributos, pseudoclases: .clase, [type], :hover
  4. Elementos y pseudoelementos: div, ::before

Cálculo de Especificidad

Cada selector recibe una "puntuación" con formato (a, b, c, d), donde:

  • a: Estilos en línea
  • b: IDs
  • c: Clases, atributos y pseudoclases
  • d: Elementos y pseudoelementos
Selector Especificidad Comentarios
p0,0,0,1Un solo elemento
.clase0,0,1,0Una clase
#id0,1,0,0Un ID
div p.clase0,0,1,22 elementos, 1 clase
p#id0,1,0,1ID + elemento
style="..."1,0,0,0Estilo en línea

Selectores Combinados

Los selectores combinados permiten seleccionar elementos en relación con otros. Aquí están los principales tipos:

Combinador Ejemplo Descripción
Descendiente ( ) div p Selecciona todos los <p> que están dentro de un <div>, sin importar el nivel.
Hijo directo (>) div > p Selecciona solo los <p> que son hijos directos de un <div>.
Hermanos adyacentes (+) h2 + p Selecciona el <p> inmediatamente después de un <h2>.
Hermanos generales (~) h2 ~ p Selecciona todos los <p> que son hermanos de un <h2>, pero no necesariamente inmediatos.
