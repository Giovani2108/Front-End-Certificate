10 - Efectos con transform, transition y animation

¿Qué es transform?

Sirve para cambiar la forma, tamaño o posición de un elemento en la pantalla. Puedes rotarlo, moverlo, escalarlo o sesgarlo.

Ejemplos básicos de transform


.sample-class {
  transform: rotate(60deg);
}
    

.sample-class {
  transform: none;
}

Funciones transform y sus variaciones

FunciónDescripciónEjemplo
matrix() Transformación compleja con 6 valores transform: matrix(1, 2, 3, 4, 5, 6);
rotate() Gira el elemento en grados transform: rotate3d(3, 2, 1, 100deg);
translate() Mueve el elemento en x, y, z transform: translate3d(10px, 20px, 30px);
scale() Cambia el tamaño del elemento transform: scale3d(2, 1, 0.3);
skew() Sesga el elemento en ángulo transform: skew(100deg);

Valores globales para transform y su uso

Estos valores controlan cómo se hereda o reinicia la propiedad transform:

  • inherit: El elemento toma el valor de transform de su elemento padre.
  • initial: Se restablece el valor a su valor por defecto, que es none.
  • revert: Revierte la propiedad al valor establecido por la hoja de estilo del usuario o navegador (por ejemplo, si una regla anterior la modificó).
  • revert-layer: Similar a revert, pero para capas de estilos específicas en cascada (CSS Layers).
  • unset: Actúa como inherit si la propiedad es heredable o como initial si no lo es. En este caso, transform no es heredable, así que equivale a initial.

    .sample-class {
    transform: inherit;      /* Hereda la transformación del padre */
    }
    .sample-class {
    transform: initial;      /* Reinicia la transformación a 'none' */
    }
    .sample-class {
    transform: revert;       /* Revierte a la regla CSS previa o del navegador */
    }
    .sample-class {
    transform: revert-layer; /* Reversión en capas CSS específicas */
    }
    .sample-class {
    transform: unset;        /* Hereda o reinicia según la propiedad sea heredable */
    }

Múltiples transformaciones juntas

Puedes combinar varias transformaciones en la misma regla:


.sample-class {
  transform: rotate(45deg) scale(1.5) translate(45px);
}

Propiedad transform-origin

Define el punto de anclaje para la transformación:


.sample-class {
  transform-origin: 10px 10px;
}
.sample-class {
  transform-origin: right bottom;
}

Propiedad transition

Permite animar suavemente cambios de propiedades.

Sintaxis shorthand


transition: property duration timing-function delay;

Ejemplo:


transition: margin-left 2s ease-in-out 0.5s;

Propiedad animation y @keyframes

animation combina muchas sub-propiedades para crear animaciones complejas.

Sintaxis shorthand


animation: name duration timing-function delay iteration-count direction fill-mode play-state;

Ejemplo básico:


.sample-class {
  animation: none 2s ease 0.5s 4 normal none running;
}

Ejemplo @keyframes


@keyframes mymove {
  from { bottom: 0px; }
  to { bottom: 100px; }
}

Uso de porcentajes en @keyframes


@keyframes mymove {
  0%, 100% { background-color: blue; }
  50% { background-color: green; }
}

Múltiples animaciones

Puedes usar varias animaciones en un mismo elemento separándolas por coma:


#some-class {
  animation: animation-a 2s linear infinite alternate,
             animation-b 3s ease infinite alternate;
}
