07 - Efectos de Texto en CSS

Propiedades Principales de Estilo de Texto

Las siguientes propiedades permiten modificar el aspecto del texto, ya sea para mejorar su legibilidad, estética o énfasis visual:

Propiedad Valores Descripción Ejemplo
text-transform none, uppercase, lowercase, capitalize Transforma el texto a mayúsculas, minúsculas o capitalización por palabra. texto en mayúsculas
font-style normal, italic, oblique Aplica estilos inclinados o cursivos al texto. Texto en cursiva
font-weight normal, bold, bolder, lighter, 100–900 Controla el grosor del texto. Texto en negrita
text-decoration none, underline, overline, line-through Subraya o tacha el texto según lo indicado. Texto subrayado
text-decoration-style solid, double, dotted, dashed, wavy Define el estilo visual de la decoración aplicada (ej. subrayado ondulado). Subrayado ondulado
text-shadow offset-x offset-y blur color Agrega sombra al texto. Texto con sombra
text-align left, right, center, justify Alinea el texto horizontalmente. Texto centrado
text-indent medida (px, em, %) Indenta la primera línea de un párrafo. Este texto tiene sangría en la primera línea.
text-overflow clip, ellipsis Controla cómo se muestra el desbordamiento del texto. Texto largo que no cabe completamente en el contenedor.
text-combine-upright none, all Combina caracteres verticalmente en un solo glifo (útil en idiomas como japonés). 2019年
text-emphasis color style símbolo Agrega énfasis visual al texto (funciona en vertical). 強調

Propiedades Avanzadas de Texto

Estas propiedades ayudan a controlar el flujo, orientación, y comportamiento del texto en su contenedor.

Propiedad Valores Descripción
word-wrap normal, break-word Permite romper palabras largas en múltiples líneas.
word-break normal, break-all, keep-all Determina cómo se dividen las palabras al final de una línea.
writing-mode horizontal-tb, vertical-rl, vertical-lr Establece la dirección del texto (horizontal o vertical).
text-justify auto, inter-word, inter-character Controla la forma de justificar el texto cuando se usa text-align: justify.
text-underline-position auto, under, left, right Posiciona la línea subrayada respecto al texto.
