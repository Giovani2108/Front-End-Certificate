09 - Animaciones CSS con @keyframes

¿Qué es @keyframes?

@keyframes es una regla en CSS que te permite crear animaciones paso a paso. Puedes cambiar propiedades (como tamaño, color, posición) desde un estado inicial hasta uno final, y más.

Piensa en mover un objeto de un lugar a otro o cambiar su color poco a poco, ¡todo eso con animaciones suaves!

Sintaxis básica

@keyframes nombreAnimacion {
  from {
    propiedad: valor-inicial;
  }
  to {
    propiedad: valor-final;
  }
}

from es el inicio de la animación y to el final. También puedes usar porcentajes para más pasos: 

@keyframes nombreAnimacion {
  0% { /* inicio */ }
  50% { /* paso intermedio */ }
  100% { /* fin */ }
}

Ejemplo práctico

Este cuadro cambia su ancho de 50px a 100px y vuelve a empezar, una y otra vez.

.box {
  width: 50px;
  height: 50px;
  background-color: lightcoral;
  animation: crecer 3s infinite ease-in-out;
}

@keyframes crecer {
  from { width: 50px; }
  to { width: 100px; }
}

Animación con varios pasos

Con porcentajes puedes hacer animaciones más complejas, cambiando varias propiedades:

@keyframes colorAlt {
  0% {
    background-color: lightcoral;
    height: 50px;
  }
  50% {
    background-color: aquamarine;
    height: 20px;
  }
  100% {
    background-color: lightcoral;
    height: 50px;
  }
}

.box-avanzada {
  animation: colorAlt 4s infinite ease-in-out;
}

Aquí el color y el tamaño cambian en diferentes etapas, ¡así tus animaciones pueden ser súper creativas!

Propiedades importantes para animaciones

Propiedad Descripción
animation-nameNombre de la animación @keyframes
animation-durationCuánto dura la animación (ej: 3s)
animation-iteration-countCuántas veces se repite (ej: infinite para infinito)
animation-timing-functionCómo va la velocidad (ej: ease, linear)
animation-directionDirección: normal, reverse, alternate

Resumen rápido

  • @keyframes define los pasos de la animación
  • animation aplica la animación a un elemento
  • Usa from/to o 0%/100% para marcar inicio y fin
  • Puedes agregar más pasos con porcentajes intermedios para animaciones más ricas
  • La animación puede repetirse, ir y volver o parar cuando quieras
