06 - Pseudoelementos y Pseudoclases

¿Qué es un pseudoelemento?

Los pseudoelementos permiten aplicar estilos a partes específicas de un elemento sin modificar el HTML. Se escriben con doble dos puntos (::). 

selector::pseudo-element {
  propiedad: valor;
}

Pseudoclases :hover y :active

Estas pseudoclases permiten aplicar estilos cuando el usuario interactúa con elementos:

  • :hover: Cuando el cursor pasa por encima.
  • :active: Mientras se hace clic o mantiene presionado.
