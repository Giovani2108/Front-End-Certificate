03 - Grid y Flexbox en CSS

¿Qué son Grid y Flexbox?

Son métodos modernos para organizar elementos en una página. Flexbox está pensado para alinear en una sola dimensión (horizontal o vertical), mientras que Grid permite organizar en dos dimensiones (filas y columnas).

Grid: propiedades básicas

Se define así:

selector {
  display: grid;
}

Contenedor Grid

PropiedadDescripción
grid-template-columnsDefine columnas (puede usar px, %, fr, repeat())
grid-template-rowsDefine filas
grid-template-areasDefine zonas con nombres
gap / row-gap / column-gapEspacio entre filas o columnas
justify-items / align-itemsAlineación de los ítems en el grid
place-itemsShorthand de las dos anteriores
justify-content / align-contentAlineación del grid dentro del contenedor

Items dentro del Grid

PropiedadDescripción
grid-column / grid-rowDefine en qué columna/fila inicia y termina el ítem
justify-self / align-selfAlineación individual dentro del espacio asignado
place-selfShorthand de los dos anteriores

Flexbox: propiedades básicas

Se define así:

selector {
  display: flex;
}

Contenedor Flex

PropiedadDescripción
flex-directionDirección: fila, columna, reverso
flex-wrapPermite que los elementos se ajusten (wrap) o no (nowrap)
justify-contentAlinea en el eje principal
align-itemsAlinea en el eje cruzado

Items Flex

PropiedadDescripción
flex-grow / flex-shrinkPermite que crezca o reduzca en proporción
flex-basisTamaño inicial del ítem
orderCambia el orden visual del ítem
align-selfAlineación individual
