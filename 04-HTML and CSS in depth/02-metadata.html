Metaetiquetas en HTML

Las etiquetas <meta> permiten incluir información adicional sobre el documento HTML, útil para navegadores, buscadores y dispositivos móviles.

1. Metaetiqueta de conjunto de caracteres

Define la codificación de caracteres de la página.

<meta charset="UTF-8">

2. Metaetiquetas básicas para SEO

<meta name="description" content="...">
Breve descripción de la página para los motores de búsqueda.
<meta name="title" content="...">
Título de la página (complementario a la etiqueta <title>).
<meta name="author" content="...">
Nombre del autor o autores del documento.
<meta name="language" content="...">
Idioma principal del contenido (por ejemplo, "español").
<meta name="robots" content="index,follow">
Indica a los motores de búsqueda que indexen y sigan los enlaces.
<meta name="googlebot" content="notranslate">
Evita que Google traduzca automáticamente la página.
<meta name="revised" content="Fecha y hora">
Última fecha de modificación del documento.
<meta name="rating" content="safe for kids">
Define la audiencia apropiada para el contenido.
<meta name="copyright" content="...">
Información de derechos de autor.

3. Metaetiquetas http-equiv

Simulan cabeceras HTTP (no recomendadas para uso moderno, pero aún comunes).

<meta http-equiv="refresh" content="30">
Recarga la página cada 30 segundos.
<meta http-equiv="content-type" content="text/html">
Define el tipo de contenido que representa el documento.
<meta http-equiv="default-style" content="...">
Indica el estilo predeterminado.
<meta http-equiv="Content-language" content="es">
Especifica el idioma del contenido.
<meta http-equiv="Cache-Control" content="no-cache">
Evita que el navegador almacene en caché la página.

4. Metaetiquetas para diseño adaptable (Responsive)

<meta name="format-detection" content="telephone=yes">
Convierte números en enlaces para llamadas telefónicas en móviles.
<meta name="HandheldFriendly" content="true">
Optimiza la visualización en dispositivos móviles antiguos.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
Escala correctamente el contenido en pantallas móviles.
